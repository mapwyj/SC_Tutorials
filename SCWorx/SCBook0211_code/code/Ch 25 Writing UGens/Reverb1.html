Reverb1 A nice reverb pseudo-ugen


Here's the usage example from Chapter 25 of the SuperCollider book:


s.boot;

(

x = {

var freq, son, out;

// Chirps at arbitrary moments

freq = EnvGen.ar(Env.perc(0, 0.1, 10000), Dust.ar(1));

son = SinOsc.ar(freq, 0, 0.1);

// We apply reverb to the left & right channels separately

out = {Reverb1.ar(son, cutoff: 2500)}.dup;

}.play(s);

)

x.free;
