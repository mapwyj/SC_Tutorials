// 4. Evaluating code


// To run (evaluate) one line of code, for example: 


Server.default.boot; // Boot the default server. 


/* To evaluate the above line, place the text cursor anywhere on that line and press the <enter> key. If you don't have an enter key, then you can use ctrl-Return, Ctrl-c, fn-Return( on Some Macs), or Shift-Return. */  


/* After booting the default server above, run the following line to start making a sound: */


{ SinOsc.ar([400, 401], 0, 0.1) * Saw.ar([11, 33], 1) * EnvGen.kr(Env.sine(10)) }.play


/* In addition to playing a sound, SuperCollider posts the following message in the post window: */


Synth("-613784702" : 1000)


/* Press command-period (cmd-.) to stop synthesis.*/


////////////////////////////////////////////////////////////////////////////////////////////////////


/* To run more than one line of code, select all the lines and press <enter>.

Or, with the cursor to the right of the top parenthesis or the left of the bottom one, press cmd-shift-b.
